MCCG

The MCCG, or Minecraft Command Generator (it's an acronym) was made by me originally in C#.
But after I learned about the possibilities that Java gives you, I decided to port it over.

It is now rather dead, not active in development. You can still download it from its GitHub page, but I don't encourage you to.

If you want to know if and when I continue to work on the MCCG, you can join my Discord Server. (only if you want)
Its main purpose is for Minecraft Datapacks, but you can still join. (you probably came from there because there's no other reason you would know this exists)
