Syntax

Lettercode has to be on one line with no spaces.
Below are the syntax letters with their equivalent in literally every other language.
Below every syntax a short example with the javascript equivalent is given.

l = ; (Has to be at the end of every statement)
Example: hsHisl (console.log('Hi');

x = End program
The program has to end with this or it will not run.
You can use it everywhere else as well, like inside if statements.

h = console.log()
Example: hsHisl
As values for variables and stuff like this, the structure [surrounder][content][surrounder] is used.

Surrounders:
- s (string) Example: sHis
- c (character) Example: cHc
- i (integer) Example: i42i
- d (double) Example: d4.2d
- b (boolean) Example: btrueb

A special case is u (user input)
It doesn't follow the same structure, it's just a u.
Use it if you want the user to input something.

Rules for string/character values:
The character 'g' is the escape character.
gn = newline
gl = whitespace
gs/gc = escape characters to not end string early
gg = escape itself

v = Variable initialization
Example: vVARei42il (let var = 42;)

Rules for variables:
Variable names must be uppercase.

e = = (Used to assign a value to variables)
Example: vVAResHisl (let var = 'Hi';)

p = + (Used for addition)
Example: vVARei4ipi2il (let var = 4 + 2;)

m = - (Used for subtraction)
Example: vVARei4imi2il (let var = 4 - 2;)

n = * (Used for multiplication)
Example: vVARei4ini2il (let var = 4 * 2;)

q = / (Used for division)
Example: vVARei4iqi2il (let var = 4 / 2;)

Start a statement with an uppercase variable name to make operations to it.

pe = += (Adds a value to a variable)
Example: VARpei42il (var += 42;)

me = -= (Subtracts a value from a variable)
Example: VARmei42il (var -= 42;)

ne = *= (Multiplies a variable with a value)
Example: VARnei42il (var *= 42;)

qe = /= (Divides a variable by a value)
Example: VARqei42il (var /= 42;)

Conditions
Conditions are used in two places: if and while statements.
To make a condition, enter a value, a conditional operator and another value. (val1 op val2)

Conditional operators:
- gt (greater than (>))
- lt (less than (<))
- et (equal to (==))

Example: i42igti7i
Returns true, because 42 is greater than 7.

Block statements

j = if
Structure: j[condition]t[code]z
t is the opening brace ({) and z the closing brace(})
Example: jVARlti42ithi42ilz
If the variable var is less than 42, it will print 42.

r = while Structure: r[condition]t[code]z
t is the opening brace ({) and z the closing brace(})
Example: vVARei1ilrVARlti43ithVARps:gl42slVARpei1ilzx
This program will print out the number 42 42 times. You can copy this into the input box and see that it works.

Small breakdown:
vVARei1il - Initializes the variable var with the value of 1
rVARlti43it...z - Loops as long as var is less than 43
hVARps:gl42sl - Print the var followed by ': 42'
VARpei1il - Increases var by one so the loop will run 42 times
x - Ends the program
